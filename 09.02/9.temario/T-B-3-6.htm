Actividad
(ver mapaFinal)
Crear una aplicación final con mapas con Leaflet. Este es un ejemplo de interface [b].
1) Crear un variable json con los datos: las coordenadas, nombre, ciudad, nombre de imagen de 10 catedrales de europa. Leer el json [a] desde js. Se proporciona un archivo json con datos de ejemplo.
2) Crear un menú con las catedrales. El menú contendrá la foto, el nombre y la ciudad de cada una.
3) Crear un mapa y cargar los marcadores.
4) Añadir un click en el menu para cada sitio para resaltar/animar el marcador correspondiente.
5) Añadir la opción de seleccionar/deseleccionar cada marcador desde el menu.
6) Añadir un boton para crear un poligono entre los marcadores seleccionados.
7) Añadir el poligono creado como otra entrada en tu menu, solicitando un nombre al usuario.
8) Mostrar en el menu alguna información del poligono ademas del nombre, como la superficie, numero de puntos...
9) Selecciona el poligono en el menu y destacar el poligono en el mapa.
10) Añadir la opción de selccionar y borrar los poligonos. Se deben borrar tanto en el menu como en el mapa.
