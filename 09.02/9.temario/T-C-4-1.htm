Actividad
(ver videosParrilla)
Hacer una parrilla de videos. Practicar los conceptos de clases, objetos, constructores, get, set, atributos

Estructura de datos ==
01) Crearemos una lista de objetos video instanciados de una clase. Crear las clases, encapsulacion de datos, set y get y constructores usando diferentes estrategias de javascript.
02) El objeto video tendra los atributos: 
- vid(cadena11),
- título(char5),
- duración(h/m/s),
- fecha de subida(dd/mm/yyyy),
- visitas(entero),
- imagen(https://i.ytimg.com/vi/uWs5lsWXLbo/mqdefault.jpg). 
03) Crea un metodo dentro de tus objetos que retorne el código HTML para imprimir la ficha. 
04) Para estilizar la ficha, usar una clase ficha y acceso por selectores a los campos de la ficha.
05) Comprueba por console.log si los atributos de tus objetos son privados y si los metodos get/set funcionan

Layout de la pagina ==
06) Crear una función que dibuje en un contenedor todas las fichas. Genera al azar las fichas entre 1 y 100 fichas.
07) Crea el layout principal para tu página [03][08][26].
08) Otra maquetacion puede ser maquetar el contenedor de fichas en horizontal con un scroll [11][27].

Paginador ==
09) Maqueta el contenedor de fichas con un paginador [27][28]
10) Estilizar la ultima pagina activada
11) Iniciar el paginador en la pagina uno

Maquetar ==
12) Añade otras forma de presentar las fichas con usar el scroll
14) Usa pureCss para tu menu, filtros, botones, selectores [05]

Filtros ==
15) Filtro "todos" , si hay paginador presentar la primera pagina
16) Filtro "primeras cinco fichas"
17) Otros filtros y/o con parametros entre un mínimo y un máximo

Ordenar ==
18) Ordenar las fichas por sus campos titulo, subida, visitas, duracion
19) Otras ordenaciones, ejemplo [31]

Closure ==
20) Crea un atributo que almacene las veces que el usuario visualizó cada ficha [17].

Ficha y sus campos ==
21) Usa para la ficha una sola clase (.ficha). Usa selectores para los contenido de ficha [04][09][10][25]. 

Modal ==
22) (ver modal)[41] Abrir un modal al hacer click en el video [12][15].
23) Dentro del modal reproducir el video donde hiciste click [14][16].
24) Simplificar el modal al maximo posible
