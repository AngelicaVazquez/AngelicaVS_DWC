Actividad
(ver imagenes) [03].
Usando la api de un banco de imagenes
- Crear una cuenta gratis, usar la api para obtener las imagenes. La api gratis tienen un limites. 
- Peticiones a la API.
- Monta una parrilla de imagenes con las urls obtenidas.
- Resaltar la primera al terminar la carga.
- Usa las teclas cursores para desplazarte por todas las imagenes.
- Resaltar la imagen si tiene el cursor.
- Click tambien da el cursor a la imagen.
- Al pulsar ENTER seleccionar la foto, cambiar algún estilo para indicarlo. 
- Al pulsar ESC eliminar la selección de la imagen.
- Mover el cursor al pulsar ENTER o ESC.
- Indicar las instrucciones al usuario.
- Descargar con una extensiones de firefox las imagenes y hacer la carga de los archivos locales en el caso de que la API no retorne las imagenes
lenguaje: .focus, .querySelectorAll("img"), datos.faces[k].urls[2][128], .tabIndex
