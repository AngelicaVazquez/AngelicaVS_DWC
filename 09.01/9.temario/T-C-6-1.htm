Actividad
(ver dragdrop)
Hacer un gestor de bookmarks usando solo nodos del DOM para almacenar los datos.
-Añadir la gestion del bookmark: listar, añadir, modificar y borrar
-Gestion de las caracteristicas del bookmark
-Añadir a cada bookmark etiquetas. Buscar por internet ejemplos del uso de etiquetas. Gestion de las etiquetas: listar, añadir, eliminar, modificar, enlazar con el bookmark.
-Usar metodos de acceso y seleccion de nodos del DOM.  
-Añadir drag and drop.
-Indicar tres ejemplos que te parezcan mas adecuados en usabilidad de la gestion de etiquetas. Pensar en la usabilidad de tu pagina al diseñar el interface hacerlo lo mas facil posible de usar, usar el diseño que consideres mas óptimo. 

Funciones:
01) Crear un nuevo bookmark sin datos
02) Eliminar un bookmark existente
03) Ver la lista de todos los bookmark existentes
04) Seleccionar un bookmark y ver sus elementos
05) Modificar elementos de un bookmark ya creado.
06) Crear un nuevo bookmark y al menos 3 elementos por ejemplo titulo, url, descripcion (..) 
.
07) Ver las etiquetas de cada bookmark
08) Borrar una etiqueta al bookmark
09) Añadir una etiqueta ya existente en el bookmark al bookmark y quedan ordenadas por nombre
.
10) Ver todas las etiquetas existentes
11) Crear una nueva etiqueta
12) Borrar una etiqueta. Comprobar que se borro en los bookmark que la tenian.
.
13) Cambiar el nombre de una etiqueta existente
14) Ver lista de bookmark para una etiqueta seleccionada
15) Lista de bookmark para varias etiquetas seleccionadas
* DW2E-dragdrop-expediente-ApellidosNombre.htm ; DIC21-2100
